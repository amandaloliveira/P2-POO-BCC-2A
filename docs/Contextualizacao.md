## Tema do Projeto

O tema do nosso projeto consiste em um **Sistema de Avaliação de Jogos**, com inspiração nos sites *Leterboxd* e *Rotten Tomatoes*.

---

## Funcionalidades

**Menu Inicial**
| ÍNDICE | FUNÇÃO DISPONÍVEL                                       |
| ------ | :-----------------------------------------------------: |
| 1      | Acessar o sistema com um usuário já existente *(Login)* |
| 2      | Cadastrar um novo usuário do tipo Avaliador             |
| 3      | Sair do Sistema *(Logout)*                              |

**Usuário Avaliador**
| ÍNDICE | FUNÇÃO DISPONÍVEL                          |
| ------ | :----------------------------------------: |
| 1      | Avaliar um Jogo                            |
| 2      | Editar Minha Avaliação                     |
| 3      | Excluir minha Avaliação                    |
| 4      | Ver Detalhes de um Jogo                    |
| 5      | Exibir Lista de Categorias                 |
| 6      | Exibir Jogos de uma Categoria              |
| 7      | Adicionar Jogo aos Favoritos               |
| 8      | Remover Jogo dos Favoritos                 |
| 9      | Ver Meus Favoritos                         |
| 10     | Alterar Meu Nome                           |
| 11     | Excluir Minha Conta                        |
| 12     | Entrar/Sair de Comunidade                  |
| 13     | Interagir em uma Comunidade (Comentar/Ver) |
| 14     | Sair do Sistema *(Logout)*                 |


**Usuário Administrador**
| ÍNDICE | FUNÇÃO DISPONÍVEL             |

## Decisões Tomadas

Nosso projeto foi desenvolvido com o objetivo de abranger os quatro pilares fundamentais da Programação Orientada à Objetos: Abstração, Herança, Encapsulamento e Polimorfismo.
Dessa maneira, abrangendo os quatro pilares, expandimos nosso código, implementando mais três categorias e mais funcionalidades em nosso projeto, além de corrigir os erros presentes nos códigos anteriores.
